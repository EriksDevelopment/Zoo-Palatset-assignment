Zoo Palatset är en informationssida för en djurpark där användaren kan:
	•	Se öppettider
	•	Utforska olika djur
	•	Läsa om restauranger och aktiviteter
	•	Hitta parkens adress via Google Maps
	•	Prenumerera på nyhetsbrev

Projektet visar användning av:
	•	Flexbox-layout
	•	Sticky navbar
	•	Hover-effekter och transitions
	•	Scroll-baserade effekter (JavaScript)
	•	Formulär med validering
	•	Tabellstruktur
	•	Inbäddad Google Maps
